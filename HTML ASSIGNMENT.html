APPLICATION

1.Application is a set of programs.

2.Program is a set of instructions which is provided to machine(Laptop,Mobile).

3.To run applications in a machine to install operating system(windows,linux).

TYPES OF APPLICATIONS

1.Stand alone application.

2.Web Application.

3.Mobile Application.

4.Distributed Application.

1.STAND ALONE APPLICATION



1.The application which is work in without internet is referred as standalone application.

2.It is also called as independent application.

3.To develop this type of application make use of corejava.

4.To run this applicationin a client machine install operating system.

5.For example Notepad,caluculator,MS.Word etc.

2.WEB APPLICATION



1.The applications which are working with the help of internet and browser is referred as web application.

2.To develop web applications make use of Advance Java (OR) web technologies.

3.To use the web applications, no need of installation in a client machine.

4. web applications are also called as websites.

5.For example flipkart, amazon etc.

2.1 BROWSER



1.Browser is an application which is developed to access web application.

2.Browsers are the Standalone applications which need to install in a client machine.

3.for example Google chrome, fire fox, brave, opera mini, micro Soft edge, internet explor.

2.2 SERVER



1.Server is a machine which act's like mediator between programmer and user.

2.programer are using server to store web aplications.

3. Users can access webaplications by sending a request a server with the help of internet and server wiu provide response to the user.

Note: server and client machine present at different Locations hence we required internet to access web applications.

COMPONENTS OF WEB APPLICATION



1.Every web application have 2 important components. 

        A.FRONT END


        B.BACKEND

2.The components which are visible to an user is refered as frontend an of an application (user interface).

3.To develop these type of components make use of "web technologies".

4. The for example: login page, home page etc.

5. The components which are hidden from the user is referred as Backend of an application.

6.To develop these components make use of core java'.

7.For example Backend procedure of login page.

TYPES OF WEB APPLICATIONS

1.There are two types of web applications 

      a) Static Web Application.


      b) Dynamic Web Application.
2.An application which is having only frontend components is known as " Static web application".

3.To develop this application make use of web technologies.

4.For example wikipedia.

5.The application which is having frontend and backend is referred as Dynamic web application.

6.To develop this type of application make use of "core java","SQL"and "web technologies(Advanced java)

7.For example Flipkart,Amazon,Gmail.

3. MOBILE APPLICATIONS



1.The application which needs to download and install in mobile phones is referred as "mobile application".

2.To develop mobile applicationsmake use of flutter technology which is introduced by Google.

3.To-Run mobile applications use Os like Android/ios.

4.For example: instagram.



4.DISTRIBUTED APPLICATIONS



1.The applications which are interconnected to each other is referred as distributed application.

2.To develop these type of applications make use of "cloud technology".

3.For example :Gmail,Jio,Amazon.
